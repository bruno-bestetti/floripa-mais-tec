    Exercício 5 JSON e LocalStorage

    Crie um objeto JavaScript simples e converta-o em uma string JSON usando JSON.stringify().
    Em seguida, salve essa string JSON no LocalStorage e, depois após atualizar a página, recupere e converta-a de volta em um objeto JavaScript usando JSON.parse().

    Para acessar ao LocalStorage:
    1. Abra DevTools (Ou inspecionar), com o botão direito do mouse no navegador, ou tecla F12 do teclado.
    2. Abra a guia: "Aplicação" no menu de navegação superior.
    3. No menu de navegação lateral clique: Armazenamento Local ou Local Storage.
    4. O conjunto "Chave-Valor" estará na porta http. Ex: http://http://127.0.0.1:5500/

    Objeto:
    usuario = { nome: "Bruno", idade: 31, cargo: "Administrador" };

    Exercício 6 Interval e Timeout

    "Interval" é uma função que permite a execução repetida de um bloco de código em intervalos regulares, especificados em milissegundos.
    É comumente usado para criar animações, atualizar informações em tempo real ou executar tarefas periódicas.

    Por outro lado, o "Timeout" é uma função que executa um bloco de código após um determinado período de tempo, também especificado em milissegundos.
    É útil quando se deseja adicionar atrasos ou agendar a execução de uma função para o futuro.
