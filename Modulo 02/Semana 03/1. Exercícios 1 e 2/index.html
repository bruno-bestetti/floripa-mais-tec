Exercício 1

Enunciado

Discorra sobre os tipos de variáveis em javascript e exemplifique com um exemplo cada tipo.

Resposta:

No JavaScript, existem vários tipos de variáveis que podem ser utilizadas para armazenar diferentes tipos de valores.
JavaScript é uma linguagem de tipagem dinâmica, o que significa que você não precisa declarar explicitamente o tipo de uma variável antes de usá-la. O tipo é inferido automaticamente com base no valor atribuído à variável.

Variável numérica: Number

As variáveis numéricas são usadas para armazenar valores numéricos, como números inteiros ou decimais.

Variável de texto: String

As variáveis de texto são usadas para armazenar sequências de caracteres. Elas são delimitadas por aspas simples ('') ou aspas duplas ("").

Variáveis booleanas: Boolean

As variáveis booleanas podem ter apenas dois valores: true (verdadeiro) ou false (falso). Elas são usadas para representar condições lógicas.

Variáveis de Array:

As variáveis de array são usadas para armazenar uma coleção de valores em uma única variável. Os elementos dentro de um array podem ser de qualquer tipo, incluindo números, strings, booleanos e até mesmo outros arrays.

Variáveis de Objetos:

As variáveis de objeto são usadas para armazenar conjuntos de pares de chave-valor, onde cada chave é uma string que identifica o valor associado a ela. Isso permite organizar e acessar dados de forma estruturada.

Declaração de variáveis JS: let | const | var

Let

O let é usado para declarar variáveis que podem ser reatribuídas posteriormente.
Essas variáveis têm escopo de bloco, o que significa que elas só são acessíveis dentro do bloco onde foram declaradas.

Dica:O let é geralmente recomendado quando você precisa alterar o valor da variável ao longo do tempo ou quando a variável precisa ter escopo limitado a um bloco específico.

Const

O const é usado para declarar variáveis ​​que não podem ser reatribuídas após a sua inicialização. Essas variáveis também têm escopo de bloco.

Dica:O const é recomendado quando você deseja definir uma constante ou um valor que não deve ser alterado. É uma boa prática usar o const sempre que possível, pois ajuda a evitar a reatribuição acidental e torna o código mais legível.

Var

O var era a forma antiga de declarar variáveis em JavaScript.
Ele também tem escopo de função, o que significa que a variável é acessível em todo o escopo da função onde foi declarada.
No entanto, diferentemente do let e do const, o var não tem escopo de bloco.
O var ainda é usado em versões mais antigas do JavaScript e pode ter comportamentos inesperados devido ao seu escopo de função.

Dica:É recomendado utilizar let ou const em vez de var na maioria dos casos, a menos que você precise especificamente do comportamento de escopo de função.

Exercício 2

Enunciado

Discorra sobre o que é uma função em Javascript e exemplifique com 3 tipos de funções que resolvem determinado problema.
Exemplo: Função para calcular IMC, recebe o peso e altura por parâmetros e retorna um resultado numérico.

Resposta:

Em JavaScript, uma função é um bloco de código que pode ser reutilizado para realizar uma tarefa específica. Ela recebe um conjunto de parâmetros, executa um conjunto de instruções e, opcionalmente, retorna um valor. As funções são fundamentais na programação, pois permitem organizar e modularizar o código, tornando-o mais legível, reutilizável e fácil de dar manutenção.

Função calculadora IMC

Essa função recebe o peso e altura como parâmetros, realiza o cálculo do IMC e retorna o resultado.
Ela é útil para calcular o índice de massa corporal de uma pessoa com base nos valores informados.

Função encontra maior valor

Essa função recebe um array como parâmetro e itera sobre ele para encontrar o maior valor presente no array.
Ela é útil quando você precisa encontrar o valor máximo em um conjunto de dados.

Função número par

Essa função recebe um número como parâmetro e verifica se ele é par.
Ela retorna true se o número for par e false caso contrário.
Essa função pode ser utilizada quando você precisa verificar a paridade de um número em seu código.
